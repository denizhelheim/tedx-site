# TEDxYeal Youth Web Sitesi

Modern ve responsive TEDx Youth etkinliği web sitesi.

## 🚀 Özellikler
- **Modern Tasarım**: Inter fontu, gradient'ler ve animasyonlarla güncel görünüm
- **Responsive Design**: Mobil uyumlu hamburger menü ile tüm cihazlarda mükemmel görünüm
- **Smooth Animasyonlar**: Scroll animasyonları, hover efektleri ve geçişler
- **İnteraktif Öğeler**: Form validasyonları, typing efektleri ve parallax
- **SEO Dostu**: Semantik HTML, meta etiketleri ve hızlı yükleme
- **Erişilebilirlik**: ARIA etiketleri ve klavye navigasyonu

## 📁 Dosya Yapısı
```
tedx-site/
├── index.html            # Ana sayfa
├── tema.html             # Tema sayfası
├── basvurular.html       # Başvuru formu
├── basvuru_gozlemci.html # Gözlemci başvurusu
├── basvuru_katilimci.html # Katılımcı başvurusu
├── basvuru_basin.html    # Basın başvurusu
├── basvuru_eglendirme.html # Eğlendirme başvurusu
├── README.md             # Bu dosya
├── assets/               # Görseller ve medya
├── css/
│   └── style.css         # Modern CSS stilleri
└── js/
    └── script.js         # JavaScript etkileşimleri
```

## 🛠️ Teknolojiler
- **HTML5**: Semantik yapı
- **CSS3**: Flexbox, Grid, Animasyonlar, Media Queries
- **JavaScript**: ES6+, Intersection Observer, DOM manipülasyonu
- **Font Awesome**: İkonlar
- **Google Fonts**: Inter font ailesi

## 🚀 Başlangıç
1. `index.html` dosyasını bir web tarayıcısı ile açın
2. Mobil cihazlarda hamburger menüyü test edin
3. Formları doldurup gönderin

## 🎨 Özelleştirme

### Renkleri Değiştirmek
`css/style.css` dosyasında `:root` bölümündeki CSS değişkenlerini düzenleyin:

```css
:root {
    --primary-color: #ff0000;
    --secondary-color: #1a1a1a;
    --accent-color: #ff4444;
}
```

### İçerik Eklemek
HTML dosyalarında ilgili bölümleri düzenleyin.

### Yeni Sayfalar Eklemek
Yeni HTML dosyaları oluşturun ve aynı CSS ve JS dosyalarını kullanın.
